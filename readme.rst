===================
CallMeNotifications
===================

Наверное многие знают об услуге "перезвони мне" мобильных операторов (в простонародье так-же называемые "попрошайками" или "бомжами"). Меня всегда раздражало то, что такие СМСки копятся во входящих, мешая искать более информативные сообщения.

Основные цели приложения
------------------------
 * Вывод сообщения о просьбе перезвонить в область уведомлений, при нажатии на которое будет набран номер абонента
 * Удаление сообщения из входящих

.. figure:: https://github.com/clericJ/callme-notifications/raw/master/screenshot.png


Поддерживаемые операторы
------------------------
 * Мегафон
 * МТС
 * Билайн
 * TELE2
 * НСС
